# IV-PAFNet_dataset

Datasets used in this study for infrared–visible vehicle detection, including the DroneVehicle and Camera-vehicle datasets.

- **DroneVehicle**: A drone-based RGB-infrared cross-modality vehicle detection dataset.  
  https://github.com/VisDrone/DroneVehicle

- **Camera-vehicle**: Infrared–visible vehicle data from the OTCBVS Benchmark Dataset Collection (Dataset 03).  
  http://vcipl-okstate.org/pbvs/bench/Data/03/download.html
